Подоляка Дана 
5 варіант
1. Зчитати дані про книги з CSV файлу. За допомогою ітераторів знайти множину всіх унікальних авторів (використати std::set<std::string>) та зберегти цей результат у JSON файл у вигляді масиву. Обробити можливі помилки. Написати unit-тести.
2. Зчитати дані про користувачів з CSV файлу (колонки: id, name, email). Зберегти їх у відповідний STL-контейнер. Перетворити ці дані у формат JSON та записати у новий файл. Реалізувати власні винятки. Написати unit-тести для перевірки логіки зчитування та конвертації.
